Diese Übersicht enthält eine fortlaufend aktualisierte Liste aller CSS-Selektoren, die Teil von Spezifikationen sind, die den W3C-Status Recommendation haben.

  • Einfache Selektoren
  • Pseudo-Elemente
  • Kombinatoren

Einfache Selektoren

Beschreibung Syntax Bedeutung Zuerst definiert in CSS-Level
Universalselektor * Selektiert jedes Element. 2
Typselektor foo Selektiert jedes Element vom Typ foo. 1
Klassenselektor .foo Selektiert jedes Element mit der Klasse foo. (Äquivalent zu [class~=foo].) 1
ID-Selektor #foo Selektiert jedes Element mit der ID foo. (Äquivalent zu [id=foo].) 1
Attributselektor [foo] Selektiert jedes Element, bei dem das Attribut foo gesetzt ist (unabhängig vom Wert). 2
[foo=bar] Selektiert jedes Element mit dem Attribut foo, dessen Wert genau bar ist. 2
[foo~=bar] Selektiert jedes Element mit dem Attribut foo, dessen Wert eine Liste von durch Leerzeichen getrennten Zeichenketten ist, von denen eine Zeichenkette genau bar ist (z. B. bar oder bar baz, aber nicht barbaz). 2
[foo^=bar] Selektiert jedes Element mit dem Attribut foo, dessen Wert genau mit bar beginnt. 3
[foo$=bar] Selektiert jedes Element mit dem Attribut foo, dessen Wert genau mit bar endet. 3
[foo*=bar] Selektiert jedes Element mit dem Attribut foo, dessen Wert die Zeichenfolge bar enthält. 3
[foo|=bar] Selektiert jedes Element mit dem Attribut foo, das eine Liste von durch Bindestriche getrennten Werten enthält, die mit bar beginnt (z. B. bar oder bar-baz). 2
Pseudo-Klasse :root foo:root Selektiert das Wurzel-Element eines Dokuments, in dem sich foo befindet. In HTML ist es immer das html-Element. 3
Pseudo-Klasse :nth-child() foo:nth-child(a) Selektiert das Element foo, das das a-te Kindelement eines Elternelements ist. 3
Pseudo-Klasse :nth-last-child() foo:nth-last-child(a) Selektiert das Element foo, das das a-te Kindelement eines Elternelements ist, wobei die Zählung beim letzten Element beginnt. 3
Pseudo-Klasse :first-child foo:first-child Selektiert das Element foo, das das erste Kindelement eines Elternelements ist. (Äquivalent zu foo:nth-child(1).) 2
Pseudo-Klasse :last-child foo:last-child Selektiert das Element foo, das das letzte Kindelement eines Elternelements ist. (Äquivalent zu foo:nth-last-child(1).) 3
Pseudo-Klasse :only-child foo:only-child Selektiert das Element foo, das das einzige Kindelement eines Elternelements ist. (Äquivalent zu :first-child:last-child.) 3
Pseudo-Klasse :nth-of-type() foo:nth-of-type(a) Selektiert das a-te Element vom Typ foo, das ein Kindelement eines Elternelements ist. 3
Pseudo-Klasse :nth-last-of-type() foo:nth-last-of-type(a) Selektiert das a-te Element vom Typ foo, das ein Kindelement eines Elternelements ist, wobei die Zählung beim letzten Element dieses Typs beginnt. 3
Pseudo-Klasse :first-of-type foo:first-of-type Selektiert das erste Element vom Typ foo, das ein Kindelement eines Elternelements ist. (Äquivalent zu foo:nth-of-type(1).) 3
Pseudo-Klasse :last-of-type foo:last-of-type Selektiert das letzte Element vom Typ foo, das ein Kindelement eines Elternelements ist. (Äquivalent zu foo:nth-last-of-type(1).) 3
Pseudo-Klasse :only-of-type foo:only-of-type Selektiert das Element foo, das das einzige Kindelement vom Typ foo eines Elternelements ist. (Äquivalent zu :first-of-type:last-of-type.) 3
Pseudo-Klasse :empty foo:empty Selektiert jedes Element foo, das keine Kindelemente (einschließlich Text) besitzt. 3
Pseudo-Klassen :link und :visited foo:link Selektiert jedes Element foo, das ein Verweis ist, der noch nicht besucht wurde. 1
foo:visited Selektiert jedes Element foo, das ein Verweis ist, der besucht wurde. 1
Pseudo-Klassen :hover, :active und :focus foo:hover Selektiert das Element foo, auf das der Benutzer gerade zeigt. 2
foo:active Selektiert das Element foo, das gerade durch den Benutzer aktiviert wird, z. B. in der Zeit zwischen dem Drücken und dem Loselassen der Maustaste. 1
foo:focus Selektiert das Element foo, das gerade den Fokus hat, z. B. durch die Tabulator-Taste. 2
Pseudo-Klasse :target foo:target Selektiert das Element foo, das Ziel eines Verweises ist. 3
Pseudo-Klassen :enabled, :disabled foo:enabled Selektiert jedes Element foo, das aktiviert ist. 3
foo:disabled Selektiert jedes Element foo, das deaktiviert ist. 3
Pseudo-Klasse :checked foo:checked Selektiert jedes Element foo, das gerade markiert ist. In HTML sind das Radiobutton- und Checkbox-Elemente. 3
Pseudo-Klasse :lang() foo:lang(bar) Selektiert jedes Element foo, das in der Sprache bar ist. (:lang(bar) ist nicht äquivalent zu [lang|=bar]!) 2
Pseudo-Klasse :not() foo:not(bar) Selektiert jedes Element foo, auf das der einfache Selektor (außer :not() selbst) bar nicht zutrifft. (:not(:not(bar)) ist demnach nicht erlaubt.) 3

Pseudo-Elemente

Beschreibung Syntax Bedeutung Zuerst definiert in CSS-Level
Pseudo-Element ::first-line foo::first-line Selektiert die erste formatierte Zeile eines Textes eines Elements foo. 1
Pseudo-Element ::first-letter foo::first-letter Selektiert den ersten formatierten Buchstaben eines Textes eines Elements foo. 1
Pseudo-Elemente ::before und ::after foo::before Erzeugt ein Element vor einem Element foo und selektiert es. 2
foo::after Erzeugt ein Element nach einem Element foo und selektiert es. 2

Kombinatoren

Beschreibung Syntax Bedeutung Zuerst definiert in CSS-Level
Nachfahrenkombinator foo bar Selektiert alle Elemente bar, die Nachfahren vom Element foo sind. 1
Kindkombinator foo > bar Selektiert alle Elemente bar, die ein Kind des Elements foo sind. 2
Geschwisterkombinator (adjazent) foo + bar Selektiert alle Elemente bar, die einen unmittelbaren Vorgänger foo auf gleicher Ebene haben. 2
Geschwisterkombinator (allgemein) foo ~ bar Selektiert alle Elemente bar, die einen mittelbaren Vorgänger foo auf gleicher Ebene haben. 3
